## ğŸš€ Getting Started

### Prerequisites

- **Node.js** (v14 or higher) - [Download here](https://nodejs.org/)
- **npm** - Package manager (comes with Node.js)
- **OpenAI API Key** - [Get one here](https://platform.openai.com/account/api-keys) with billing enabled
- **Modern web browser** - Chrome, Firefox, Safari, or Edge

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/learnbloom-ai.git
   cd learnbloom-ai
   ```

2. **Set up environment variables:**
   ```bash
   cp .env.example .env
   ```
   
   Edit `.env` and add your OpenAI API key:
   ```env
   OPENAI_API_KEY=sk-your-actual-openai-api-key-here
   PORT=5000
   NODE_ENV=development
   ```

3. **Install backend dependencies:**
   ```bash
   cd server
   npm install
   ```

4. **Install frontend dependencies:**
   ```bash
   cd ../client
   npm install
   ```

### Running Locally

1. **Start the backend server:**
   ```bash
   cd server
   npm run dev
   ```
   Server will run on `http://localhost:5000`

2. **Start the frontend (in a new terminal):**
   ```bash
   cd client
   npm start
   ```
   Frontend will run on `http://localhost:3000`

3. **Verify everything works:**
   - Visit `http://localhost:3000` - Main application
   - Visit `http://localhost:5000/health` - Server health check
   - Visit `http://localhost:5000/api/test-openai` - OpenAI connection test

## ğŸš¨ Troubleshooting

### Common Issues

**1. "Failed to generate plan" Error**
- **Check OpenAI API Key**: Visit `http://localhost:5000/api/test-openai`
- **Verify Billing**: Ensure your OpenAI account has billing enabled
- **Check Quota**: You might have exceeded your API usage limits
- **Server Logs**: Check backend console for detailed error messages

**2. CORS Errors**
- **Backend Running**: Make sure server is running on port 5000
- **Deployment URLs**: Verify frontend can reach backend URL
- **Browser Cache**: Clear browser cache and try again

**3. 504 Timeout Errors**
- **Large Requests**: Try shorter duration (1-2 weeks) to test
- **Server Overload**: Wait a moment and try again
- **OpenAI Rate Limits**: API might be temporarily rate-limited

**4. Data Not Saving**
- **LocalStorage Full**: Clear browser data or use "Clear all data" button
- **Browser Settings**: Ensure localStorage is enabled in browser
- **Private Mode**: Some browsers limit localStorage in private/incognito mode

**5. Vercel Deployment Issues**
- **Environment Variables**: Ensure OpenAI key is set in Vercel dashboard
- **Build Errors**: Check Vercel deployment logs for errors
- **CORS Configuration**: Verify backend allows frontend domain
